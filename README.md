# 머니메이커스 시황 데스크 (Money Makers)

머니메이커스 본부장을 위한 **AI 기반 일일 시황 및 관련주 보고서 자동 생성기**입니다.  
사용자가 입력한 실제 데이터(뉴스, 등락률 등)를 기반으로 작동하며, 환각(Hallucination) 방지와 철저한 데이터 검증(Validator) 로직이 적용되어 있습니다.

## 🚀 주요 기능 (v1)
- **증거 우선 (Evidence First):** AI가 제공된 시장 데이터를 기반으로만 분석하며, 상상에 의한 수치 창작을 차단합니다.
- **순수 JSON 응답 강제:** Gemini API 호출 시 `application/json` 포맷을 강제하여 데이터 파싱 에러를 방지합니다.
- **결정론적 자체 검증기 (Validator):** 응답받은 종목의 '시장 구분(KOSPI/KOSDAQ)', '입력 종목과 일치 여부', '수치 조작 여부' 등을 자바스크립트로 사전 검증한 뒤 마크다운으로 렌더링합니다.

---

## 💻 사용 방법

### 1. 파이썬 데이터 수집 환경 세팅
KRX(한국거래소)에서 전 종목 등락률 데이터를 안정적으로 수집하기 위한 세팅입니다.

```bash
# 필수 라이브러리 설치 (numpy 버전 충돌 방지)
pip install "numpy<2" pykrx pandas
```

### 2. 시장 데이터 추출
매일 장 마감 후 아래 스크립트를 실행하여 당일의 시장 데이터(KOSPI/KOSDAQ TOP 상승률 종목)를 가져옵니다.

```bash
python collect_market.py
```
*(성공 시 콘솔에 TOP 30 종목 텍스트가 출력됩니다. 이 텍스트를 복사해둡니다.)*

### 3. 웹 리포트 생성기 실행
해당 폴더에서 로컬 웹 서버를 띄운 후 브라우저로 접속합니다.
```bash
# 로컬 웹 서버 실행
python -m http.server 8080
```
- 브라우저에서 `http://localhost:8080` 에 접속합니다.
- 복사해둔 시장 데이터와 주요 뉴스를 붙여넣습니다.
- **[보고서 생성하기]**를 누르면 자동으로 텍스트를 검증한 후 완벽한 마크다운 보고서가 출력됩니다.

> **보안 안내:** Gemini API 키는 로컬 브라우저(`localStorage`)에만 저장되며, 절대 외부 서버로 전송되지 않습니다. 깃허브 코드 상에 하드코딩하지 마세요.
