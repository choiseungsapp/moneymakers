# 머니메이커스 시황 데스크 (Money Makers)

머니메이커스 본부장을 위한 **AI 기반 일일 시황 및 관련주 보고서 자동 생성기**입니다.
사용자가 입력한 실제 데이터(뉴스, 등락률 등)를 기반으로만 작동하며, 환각(Hallucination) 방지와 결정론적 데이터 검증(Validator) 로직이 적용되어 있습니다.

## 🚀 주요 기능

- **증거 우선 (Evidence First):** AI가 제공된 시장 데이터를 기반으로만 분석하며, 상상에 의한 수치·관계 창작을 차단합니다.
- **탑다운 분석 구조:** 거시/뉴스 → 섹터/테마 → 종목 → 파생으로 내려가는 인과 사슬을 강제합니다.
- **순수 JSON 응답 강제:** Gemini API 호출 시 `application/json` 포맷을 강제하여 파싱 에러를 방지합니다.
- **결정론적 Validator:** 응답 종목의 '시장 구분(KOSPI/KOSDAQ)', '입력 목록 일치', '수치 일치', '파생주 근거(evidence) 입력 포함 여부'를 자바스크립트로 검증한 뒤, **검증 실패 종목은 본문에서 제외**하고 통과 종목만 렌더링합니다.
- **XSS 차단:** 모델 출력 마크다운을 DOMPurify로 sanitize한 뒤 렌더링합니다.

## 💻 사용 방법

### 1. 데이터 수집 환경 세팅

네이버 금융에서 상승률 데이터를 수집하기 위한 라이브러리만 설치하면 됩니다. (pykrx / pandas / numpy 불필요)

```bash
pip install requests beautifulsoup4
```

### 2. 시장 데이터 추출

장 마감(15:30) 이후 아래 스크립트를 실행하면 당일 최종 데이터를 가져옵니다. (장중 실행 시 실시간 값)

```bash
python collect_market.py
```

수집 내용:
- **네이버 금융 상승 페이지**에서 KOSPI + KOSDAQ 통합 **상승률 TOP 30**
- KOSPI / KOSDAQ **지수 종가 및 등락률** (best-effort, 실패 시 직접 입력 안내)

실행하면 콘솔에 결과가 출력되고, 같은 폴더에 `market_data_YYYY-MM-DD.txt` 로도 저장됩니다.

- `[상승률 TOP 30]` 블록 → 앱의 **'특징주 / 상승률 TOP'** 칸에 붙여넣기
- `[지수]` 등락률 → 앱의 **KOSPI / KOSDAQ 등락률** 칸에 입력
- 상승 사유는 비워두면 AI가 입력된 뉴스와 엮어 채웁니다.

> 출력 형식(`N. 종목명 (KOSPI|KOSDAQ) +X.XX%`)은 앱 내부 검증기(`parseMoverRows`)의 파싱 규칙과 1:1로 맞춰져 있습니다. 블록을 통째로 붙여넣어도 종목 라인만 인식되니 안전합니다.

### 3. 웹 리포트 생성기 실행

해당 폴더에서 로컬 웹 서버를 띄운 후 브라우저로 접속합니다.

```bash
python -m http.server 8080
```

- 브라우저에서 `http://localhost:8080` 접속
- 수집한 시장 데이터와 주요 뉴스를 붙여넣기
- **[보고서 생성하기]** → 자동 검증 후 마크다운 보고서 출력 (검증 실패 종목은 제외, 상단 배너에 제외 사유 표기)

## 🔐 API 키 / 보안

- Gemini API 키는 브라우저 `localStorage`에 저장되며, **보고서 생성을 위한 Gemini API 호출에만** 사용됩니다. 별도의 앱 서버에는 저장하거나 전송하지 않습니다.
- (선택) 같은 폴더에 `config.js` 를 두고 `const CONFIG = { GEMINI_API_KEY: "..." };` 형태로 키를 넣으면 실행 시 자동 입력됩니다. **`config.js` 는 절대 커밋하지 마세요 (`.gitignore` 에 반드시 추가).**
- 코드에 API 키를 하드코딩하지 마세요. 이미 공개 레포에 키가 노출된 적이 있다면 해당 키는 **즉시 폐기/재발급**을 권장합니다.
